Frequently Asked Questions

Have a question? We have the answer.

How do I find a tutor?

Simply post a request for a CITS unit you want to get tutored in, and a tutor will respond to the request in a timely manner.

How do I become a tutor?

If you are interested in becoming a tutor, simply go to the "Accept a tutoring request" page, and accept a request.

This website is run entirely by students, so getting tutored is free of charge.

What subjects are covered by the tutors?

Every and all CITS units are covered. If you require tutoring in units unrelated to CITS, you can still request it.
